CancellableTask.getCancellationToken ()

Full Usage: CancellableTask.getCancellationToken ()

Parameters:
    () : unit

Returns: CancellableTask<CancellationToken> The default CancellationToken.

Gets the default cancellation token for executing computations.

() : unit
Returns: CancellableTask<CancellationToken>

The default CancellationToken.

Example 

 use tokenSource = new CancellationTokenSource()
 let primes = [ 2; 3; 5; 7; 11 ]
 for i in primes do
     let computation =
         cancellableTask {
             let! cancellationToken = CancellableTask.getCancellationToken()
             do! Task.Delay(i * 1000, cancellationToken)
             printfn $"{i}"
         }
     computation tokenSource.Token |> ignore
 Thread.Sleep(6000)
 tokenSource.Cancel()
 printfn "Tasks Finished"
val tokenSource: obj
val primes: int list
val i: int
val computation: (obj -> obj)
val printfn: format: Printf.TextWriterFormat<'T> -> 'T
val ignore: value: 'T -> unit
This will print "2" 2 seconds from start, "3" 3 seconds from start, "5" 5 seconds from start, cease computation and then followed by "Tasks Finished".
